# Tampermonkey 油猴脚本合集

个人收集和编写的 Tampermonkey 用户脚本，解决日常工作中的效率问题。

## 脚本列表

| 目录 | 脚本 | 说明 |
|------|------|------|
| [致远学院](./致远学院/) | CoolCollege 作答详情解锁 | 解锁酷学院考试数据页面的灰色「作答详情」按钮 |

## 安装方式

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击对应脚本目录下的 `.user.js` 文件
3. 点击 Raw 按钮即可安装
