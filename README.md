# Tampermonkey 油猴脚本合集

个人收集和编写的 Tampermonkey 用户脚本，解决日常工作中的效率问题。

## 脚本列表

| 目录 | 脚本 | 说明 |
|------|------|------|
| [致远学院](./致远学院/) | [作答详情解锁](./致远学院/coolcollege-unlock-only.user.js) | 解锁灰色「作答详情」按钮 |
| | [下载试题](./致远学院/coolcollege-download-only.user.js) | 一键下载完整试题 JSON |
| | [上传题库](./致远学院/coolcollege-upload-only.user.js) | 一键上传到题库系统 |
| | [合集](./致远学院/coolcollege-unlock.user.js) | 全部3个功能合一 |

## 安装方式

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击对应脚本链接
3. 点击 Raw 按钮即可安装
